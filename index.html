CI/CD Лабораторна робота що завгодно

Цей веб-додаток було автоматично розгорнуто за допомогою GitHub Actions та Watchtower.

Інформація про проект

Цей проект демонструє налаштування безперервної інтеграції та доставки (CI/CD) з використанням:

  • AWS EC2 для хостингу
  • Docker для контейнеризації
  • GitHub Actions для автоматизації збірки
  • Watchtower для автоматичного оновлення контейнерів
